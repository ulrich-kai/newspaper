Newspaper – Blog Reader für iOS
Newspaper ist eine native App für iPhone und iPad, die deine Blogbeiträge von kaiulrich.eu übersichtlich darstellt. Eine einfache, schnelle und sichere Möglichkeit, deine Inhalte zu lesen und zu verwalten.

🔒 Transparenz & Sicherheit
Dieser Code wird hier öffentlich zur Verfügung gestellt, um vollständige Transparenz zu bieten:

	•	✅ Keine Schadsoftware – Der komplette Quellcode ist einsehbar
	•	✅ Datenschutz – Keine Nachverfolgung oder externe Datenerfassung
	•	✅ Keine Tracking – Keine Telemetrie oder Analytics
	•	✅ Open Inspection – Jeder kann den Code überprüfen
	
📱 Features

	•	Blogbeiträge anzeigen – Alle Artikel von kaiulrich.eu in einer nativen App
	•	Favoriten – Beiträge speichern und später lesen
	•	Teilen – Artikel einfach mit anderen teilen
	•	Intuitive Navigation – Optimiert für iOS und iPadOS
	•	Offline-Support – Gelesene Artikel auch ohne Internetverbindung verfügbar
	
🛠 Technologie

	•	Sprache: Swift
	•	Plattform: iOS / iPadOS
	•	Content-Quelle: kaiulrich.eu
	•	Entwicklungsumgebung: Xcode
	
📁 Repository-Struktur

	•	Contentswift.swift – Hauptimplementierung der App-Logik, UI-Components und Content-Management
	
🔍 Code-Inspektion
Du kannst die Contentswift.swift-Datei hier einsehen und überprüfen. Der Code ist bewusst offen gelegt, um:

	1.	Sicherheit und Vertrauenswürdigkeit zu demonstrieren
	2.	Interessierte Entwickler einzuladen, den Code zu reviewen
	3.	Vollständige Kontrollierbarkeit zu gewährleisten
	
💾 Installation & Verwendung

	1.	Clone dieses Repository
	2.	Öffne das Xcode-Projekt
	3.	Wähle dein iOS-Ziel (iPhone oder iPad)
	4.	Build & Run
	
⚙️ Anforderungen

	•	Xcode 15.0 oder neuer
	•	Swift 6.0 oder neuer
	•	iOS 15.0 oder neuer
	•	iPadOS 15.0 oder neuer
	•	Internetverbindung für Content-Sync (initial)
	
📰 Content-Quelle
Alle Blogbeiträge stammen von kaiulrich.eu und werden direkt in die App geladen.
